      The following are pdf's of selected emails that announce 
	the ongoing incremental functional releases. The notes
	generally contain some useful information about the
	features being released in each increment. The main
	email archive is located at:
	   http://lists.openliberty.org/pipermail/openaz/

	Probably the most useful is the first one which contains
	the download, build, and run instructions, but it also
	refers to some of the earlier emails, which are generally
	also in this list. 

	Note: some emails refer to attached docs. One should go
	to the associated link below the pdf to obtain these
	materials.
